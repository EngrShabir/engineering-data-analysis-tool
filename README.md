# Engineering Data Analysis Tool

## 📌 Description
A Python tool for analyzing engineering datasets from CSV files.

## ⚙️ Features
- Load CSV file
- Calculate mean, max, min, std
- Identify highest and lowest values
- Plot graphs and save as PNG
- Generate timestamped report

## 🛠️ Tech Used
- Python
- pandas
- matplotlib

## ▶️ Run
```bash
pip install pandas matplotlib
python main1.py
